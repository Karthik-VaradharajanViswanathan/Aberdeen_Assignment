API Automation Project Setup Notes:
i) Install java version "17.0.8" and check if its properly installed 
 java -version
ii) Install Apache Maven "3.9.5" nd check if its properly installed 
 mvn -version
iii) Set the environment variables :
 JAVA_HOME : C:\Program Files\Java\jdk-17
 MAVEN_HOME :C:\K@rt\Enhance\apache-maven-3.9.5
iv) Install Eclipse
v) Clone the repo
vi) Perform Maven Clean and Maven Install to download all required dependecies
vii) To run test as TestNG, install TestNG for eclipse plugin from the eclipse market place.
viii) Uncheck the maven configuration parameters to TestNG - Windows/Preferences/TestNG/Maven
